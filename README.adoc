= Evolution Engine Action
:toc:

A generic GitHub Action to automate code evolution using the link:https://github.com/AMDphreak/evolution-engine[Evolution Engine] and a specified ruleset.

== Usage

Add the following to your GitHub Workflow:

[source,yaml]
----
- name: Evolve Code
  uses: AMDphreak/evolution-engine-action@main
  with:
    from: '5.15'
    to: '6.0'
    rules-repo: 'AMDphreak/evolution-rules-qt'
----

== Inputs

[cols="1,2,1"]
|===
| Input | Description | Default

| `path` | Path to the code to adapt | `.`
| `from` | Starting version/context | (Required)
| `to` | Target version/context | (Required)
| `rules-repo` | Repository containing the ruleset | `AMDphreak/evolution-rules-qt`
| `rules-dir` | Path to rules within the rules repo | `rules/qt`
| `extensions` | Comma-separated list of extensions | `.py,.cpp,.h`
|===
