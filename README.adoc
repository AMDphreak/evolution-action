= Evolution Engine Action
:toc:

A generic GitHub Action to automate code evolution using the link:https://github.com/AMDphreak/evolution-engine[Evolution Engine] and a specified ruleset.

== Usage

Add the following to your GitHub Workflow:

[source,yaml]
----
- name: Evolve Code
  uses: AMDphreak/evolution-engine-action@v1.1.0
  with:
    from: '5.15'
    to: '6.0'
    library: 'python/PyQt'
    in-place: 'true'
----

== Inputs

[cols="1,2,1"]
|===
| Input | Description | Default

| `path` | Path to the code to adapt | `.`
| `library` | Library/Binding subpath (e.g., `python/PyQt`) | `""`
| `from` | Starting version/context | (Required)
| `to` | Target version/context | (Required)
| `rules-repo` | Repository containing the ruleset | `AMDphreak/evolution-rules-code`
| `rules-dir` | Path to rules within the rules repo (or URL) | `rules`
| `in-place` | Modify files in-place? | `true`
| `out-dir` | Output directory for transformed files | `""`
| `extensions` | Comma-separated list of extensions | `.py,.cpp,.h`
|===

== Dual Release Strategy

To maintain both original and evolved versions, use the `out-dir` input:

[source,yaml]
----
- name: Evolve Code
  uses: AMDphreak/evolution-engine-action@v1.1.0
  with:
    from: '5.15'
    to: '6.0'
    library: 'python/PyQt'
    out-dir: 'dist/qt6'
----
